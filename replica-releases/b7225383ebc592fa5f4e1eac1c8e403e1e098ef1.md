## Other changes:
* [`b7225383e`](https://github.com/dfinity/ic/commit/b7225383e) Core Protocol: This PR unifies page instruction charging. The deterministic memory tracker (DMT) now charges for all combinations of {heap, stable} x {read, write} page access types. The cost is uniform and increased to 5000 instructions per OS page (4KiB).
